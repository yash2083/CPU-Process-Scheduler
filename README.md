# CPU Process Scheduler

**Interactive Virtual Lab for CPU Scheduling Algorithms**

This project is a web-based simulator designed to visualize and understand different CPU scheduling algorithms. It provides an interactive interface to test and observe how processes are managed by the CPU using Gantt charts and performance metrics.

## Features
- **Round Robin (RR) Simulation**: Allocates a fixed time slice to each process, ensuring fair execution in a cyclic manner.
- **Priority Scheduling Simulation**: Executes processes based on priority levels, with higher priority tasks running before lower priority ones.
- **Interactive UI**: Clean, modern interface built with HTML, CSS, JavaScript, and Tailwind CSS.
- **Visual Gantt Charts**: Dynamic visualization of CPU allocation over time.

## Technologies Used
- HTML5
- Vanilla CSS & Tailwind CSS for styling
- Vanilla JavaScript for algorithm simulation and DOM manipulation

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/yash2083/CPU-Process-Scheduler.git
   ```
2. Open `index.html` in your web browser.
3. Navigate to the **Simulator** section to start testing different scheduling algorithms.
